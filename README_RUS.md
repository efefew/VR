# VR
В этой игре надо рисовать указательным пальцем правой руки (я специально это поставил, как свою задачу). Можно нажать на кнопки в UI, чтобы сохранять, загружать последний рисунок (в списке сохранённых рисунках смысла нет, так как это тренировочный проект). Есть кнопки очистки рисунка и изменения цвета на красный и синий.

## About

Я изучил пакет metaXR для VR:
* режим passthrough
* отслеживание движения пальцев рук и контроллеров
* работа с UI в VR
